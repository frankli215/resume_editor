# 简历编辑器 (Resume Editor)

一个基于Vue 3 + Pinia的在线简历编辑器，支持实时预览、模块化管理和拖拽排序等功能。

## 功能特性

- 🎨 三栏布局设计（模块控制区、内容编辑区、实时预览区）
- 📱 响应式适配（支持宽屏、中屏、小屏设备）
- 🧩 模块化管理（基础信息、求职意向、工作经历、教育经历、项目经历、技能证书、自我评价）
- 🔄 拖拽排序（支持模块间和模块内条目拖拽）
- 🔧 富文本编辑（支持加粗、斜体、下划线、列表等格式）
- 📅 日期选择器（年月选择，支持"至今"选项）
- 👁️ 实时预览（编辑内容实时同步到预览区）
- 🎯 ATS检查（针对ATS系统优化）
- 🖨️ 打印预览（A4纸比例显示）

## 技术栈

- Vue 3 (Composition API)
- Pinia (状态管理)
- Vite (构建工具)
- CSS3 (样式)

## 项目结构

src/

## 安装与运行

### 环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

默认情况下，开发服务器会在以下地址运行：
- http://localhost:5173/ (如果5173端口被占用，会自动尝试其他端口)

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. 左侧模块控制区可以开启/关闭各个简历模块
2. 中间内容编辑区可以编辑各个模块的内容
3. 右侧实时预览区会实时显示简历效果
4. 支持拖拽排序模块和条目
5. 支持富文本编辑和日期选择
